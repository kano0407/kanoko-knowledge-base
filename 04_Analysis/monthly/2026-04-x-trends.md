## 4月のテーマ3本柱

### 📚 テーマ1：「教養×エンタメ」のクイズ＆参加型企画

**コンセプト**  
2026年のVTuberトレンド：複雑なテーマ（業界知識、新技術、ビジネストレンド）を、チャットクイズ・リアルタイム質問で「楽しく学べる」形に変換。視聴者が「へぇ」「なるほど」を体験できる環境構築。

**配信企画案**
1. **「〇〇業界クイズ大会」** — VTuber界隈・ゲーム業界・自己成長テーマで、選択肢クイズ。正解時は実況、不正解時は詳細解説。
2. **「視聴者がコメントで教えてくれる企画」** — Kanoko が「これ何だろう？」と疑問提示 → コメントで視聴者が解説。知識が誤っていたら訂正する参加型。
3. **「新年度ビジネスマナー」** — 4月は新年度・転職シーズン。新社会人向けのマナー・スキル解説を、Kanoko 流ユーモアで提示。

**X投稿キーワード（週の投稿軸）**  
`#クイズ投稿` `#本当の知識` `#一緒に学ぶ`

**投稿例**
```
【朝投稿 9:00】知識系まとめ
「VTuber業界、実はこんなテクノロジーが当たり前になってて。
ニューラルトラッキングって知ってました？」

【昼投稿 12:00】問いかけ型
「新年度、ビジネスマナーで失敗した人いますか？
正直、新社会人の時ぼく〇〇で失敗したんです」
```

### 🎮 テーマ2：「成長ドキュメンタリー」チャレンジ企画

**コンセプト**  
VTuber人気の理由の最大のポイント：「配信者が何かに挑戦・習得する『過程』を見える化」。セッション内で習得段階（困難、試行錯誤、達成）をリアルタイム実況。視聴者は「共成長」の立場で応援する。

**配信企画案**
1. **「新スキル習得配信」** — プログラミング、デザイン、データ分析など。「今日習得するぞ」と宣言 → 困難を乗り越える姿を見える化。
2. **「ゲーム攻略＆視聴者サポート」** — 難しいステージに挑戦 → コメント欄で「ここはこうやる」と視聴者がサポート。協力感が生まれる。
3. **「チャレンジ企画（4週連続）」** — 「4月末までに〇〇を習得する」と目標設定。週ごとに進捗報告。成功→失敗→リトライの全過程が信頼感を生む。

**X投稿キーワード（週の投稿軸）**  
`#習得タイムラプス` `#セッションチャレンジ` `#応援配信`

**投稿例**
```
【昼投稿 12:00】問いかけ型
「配信を見るなら、成功している人より『挑戦している人』の方が応援したくなりませんか？
私は失敗の過程が全部好きです」

【夜投稿 18:00】配信告知
「今夜の配信、◎◎を習得する配信です。
多分失敗します。でも一緒に乗り越えましょう」
```

### 🤝 テーマ3：UGC活用「視聴者ネタの発掘＆知識まとめ」

**コンセプト**  
インターネット上の親近感あるVTuberファン層は、SNSで視聴者が「自分の成長」をシェア。このUGC（ユーザー生成コンテンツ）を配信で紹介・分析し、キャンペーション化。「自分のコンテンツが配信で取り上げられる喜び」が最強のコミュニティ形成武器。

**配信企画案**
1. **「月初ハッシュタグキャンペーン」** — `#かのこの見方` で視聴者が「今月学びたいテーマ」を募集。配信で3つピックアップして放送。
2. **「視聴者コメントをマインドマップ化」** — 「〇〇について、あなたはどう思う？」と配信で問い→ コメント数十個から共通パターン抽出 → マインドマップにして視聴者に見せる。
3. **「スキル発表イベント」** — 4月末に「視聴者が学んだスキルの発表会」。配信内でスーパーチャット機能や投票で「ベストスキル」を決める。参加者の達成感が最高。

**X投稿キーワード（週の投稿軸）**  
`#みんなの知恵` `#UGC` `#あなたの視点はどう`

**投稿例**
```
【夜投稿 19:00】問いかけ型
「あなたが今月1つだけスキルを習得できるなら、何を選びますか？
コメント欄で教えてほしい。配信で取り上げます」

【月末日投稿 13:00】共有投稿
「Aさんのコメント『〇〇視点』、Bさんの『△△考え』...
視聴者の視点でマインドマップ作ったら、こんなことが見えた」
```

## 運用ルール

1. **毎週月曜夜**：その週のテーマ分析 → `/notebooklm-x-helper` で投稿テンプレ生成
2. **毎日朝昼**：知識系 + 問いかけ型の2投稿をスケジュール投稿
3. **毎週金夜**：視聴者コメント・UGCをNotebookLMで分析 → 翌週の投稿改善
4. **月末日**：3本柱の効果測定（ブックマーク率、プロフクリック、フォロワー増加）
